# DayNightCycleSystem V0.2.1.1

## 简介
昼夜循环系统

## 现有功能
1. 根据现实时间遵循固定游戏时间比->转为游戏时间
2. 根据游戏时间控制光照: 0~12~24h -> 弱~强~弱
   1. 包括全局光照: 
      1. 强度
      2. 日夜红蓝光偏向
   2. 所有其他光源: 
      1. 反光照强度: 白天熄灭晚上点亮
